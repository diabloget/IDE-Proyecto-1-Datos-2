Detailed Description

Permite crear MemoryObjects.

Objeto MemoryObject.

Almacena propiedades de variables recibidas del servidor en formato Json como objetos MemoryObject para acceder a sus propiedades facilmente.

Permite almacenar el nombre, valor con bytes, el scope y la direccion de memoria de una instruccion.
